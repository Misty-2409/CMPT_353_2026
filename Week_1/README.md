# Week 1: Introduction to Docker with Node.js and Python

This folder introduces the **basic concepts of Docker** using two simple examples:
one with **Node.js** and one with **Python (Flask)**.

Installation Requirement
- install VS code
- install Node.js (locally) (https://nodejs.org/en/download)
- install docker (https://docs.docker.com/desktop/setup/install/windows-install/)


The goal is to help students understand:
- What Docker is
- Why containers are useful
- How Docker works with different programming languages (JS and Python)
